# 个人网站规划（游戏策划）

## 1. 目标与定位
- 目标：在 30 秒内让招聘方明确你是“可落地的系统/数值导向游戏策划”。
- 受众：游戏公司招聘经理、主策、制作人。
- 站点定位：黑白极简科技感的个人作品站（可直接部署 GitHub Pages）。

## 2. 信息架构（已按你确认的顺序）
1. 实习经历
2. 项目经历
3. 游戏经历
4. 个人技能
5. 联系方式与外链

## 3. 视觉与美术方向

### 3.1 视觉风格
- 以 `风格参考（首页自我介绍）.png` 为主参考，不走常规简历模板。
- 主基调：黑白极简 + 科技感。
- 允许加入轻微噪点/纹理，增加质感与层次。

### 3.2 设计元素
- 版式关键词：极简、尖锐、克制。
- 卡片：锐角方框（不使用圆角卡片）。
- 分割：细线分区 + 大留白，突出结构感。
- 文字：
  - 标题可偏复古（有年代感）
  - 正文线条略圆润，保证可读性
  - 数据和标签可用等宽字体增强“技术感”

### 3.3 动效策略（轻量）
- 首屏标题和核心指标渐入。
- 模块滚动时卡片轻微 reveal。
- 避免炫技动画，保持专业感和阅读效率。

## 4. 页面模块规划

### 4.1 首屏 Hero
- 姓名 + 求职目标（游戏策划）
- 一句话价值主张（系统策划/数值拆解/团队协同）
- 快捷按钮：查看实习经历、查看项目、下载简历

### 4.2 实习经历（第一主模块）
- 公司：搜狐畅游
- 时间：2025.09 - 2026.03
- 重点展示：
  - 数值体系拆解方法
  - 天赋/潜能/通用设置从 0 到 1 设计
  - 输出文档与可追踪成果

### 4.3 项目经历（第二主模块）
- 光子比赛项目
- GMTK Game Jam 项目
- TapTap 聚光灯项目
- 末典之忆（独立 Demo）
- 每个项目统一结构：背景 -> 职责 -> 方案 -> 结果 -> 外链

### 4.4 游戏经历（第三主模块）
- 不做纯清单，改为“策划视角标签化”展示：
  - 动作/ARPG
  - Roguelike
  - SLG/长线运营
  - 射击与战术竞技
  - 关卡解谜/平台交互
- 强调“如何从体验中反推系统设计”。

### 4.5 个人技能（第四主模块）
- 流程与协作工具：Excel、Git、XMind、飞书
- 引擎与实现：UE5、Unity（基础）
- 设计背景与空间能力：SU/Blender/CAD
- 英语与奖项信息

## 5. GitHub Pages 落地方案
- 采用纯静态站（HTML/CSS/JS）优先，部署简单、维护成本低。
- 发布方式：主分支直发或 `gh-pages` 分支发布。
- 首版交付（MVP）：
  - 完整首页 + 四大内容模块
  - 可点击外链区
  - 简历下载入口
  - 基础 SEO（title/description/OG）
